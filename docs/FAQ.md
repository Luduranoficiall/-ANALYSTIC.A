# FAQ — Perguntas Frequentes

**1. Como faço o deploy em produção?**
- Use `docker-compose up -d` ou `kubectl apply -f k8s/`.

**2. Como obtenho suporte?**
- Envie e-mail para contato@analytica.com.br ou use o formulário na landing.

**3. Como baixar os materiais premium?**
- Acesse a área de downloads na landing ou na pasta `/docs`.

**4. Como gerar relatórios personalizados?**
- Use o dashboard para exportar em PDF, Excel ou PPT.

**5. Como funciona a segurança dos dados?**
- Criptografia AES-256, RSA-4096, TLS 1.3, HMAC SHA-256, JWT RS256.

**6. Posso integrar com outros sistemas?**
- Sim, via API RESTful documentada.

**7. Como funciona o onboarding de novos devs?**
- Siga o [guia de onboarding](onboarding.md).

**8. Como acessar o tour interativo?**
- Veja o arquivo [tour_interativo.md](tour_interativo.md) ou acesse o menu de ajuda na landing.
